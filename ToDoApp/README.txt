# To-Do App
This is a simple to-do app that allows you to create and manage tasks.

# Features
Create new tasks with a name, description and a due date
Filter tasks by name and date range
Save tasks to local storage

# Getting Started
To use the app, simply open the index.html file in your web browser. You can then create new tasks by entering a name and a due date, and clicking the "Create" button. To change task status drag it to "In Progress" or "Done" containers. To filter tasks by name or date range, click the search icon and select name and date range.

# Technologies Used
This app was built using HTML, CSS, and Vanila JavaScript.
